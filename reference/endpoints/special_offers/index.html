List special offers

List all special offers for given account(s).

GET /special_offers
  • OAuth Scopes:
  • public

Response

  "links": {
    "special_offers.rental": "https://www.bookingsync.com/api/v3/rentals/{special_offers.rental}"
  },
  "special_offers": [
    {
      "links": {
        "rental": 1
      },
      "id": 1,
      "name": {
        "en": "Special offer for 1"
      },
      "start_at": "2014-08-15T22:42:33Z",
      "end_at": "2014-08-22T22:42:33Z",
      "discount": 10,
      "created_at": "2014-08-08T22:42:33Z",
      "updated_at": "2014-08-08T22:42:33Z"
    }
  ]
}

Create a new special offer

Creates a special offer for given rental.

POST /rentals/:rental_id/special_offers

Parameters

Name Type Description
name String Required. Special Offer’s name.
start_at Date Required. Special Offer’s start date.
end_at Date Required. Special Offer’s end date.
discount Integer Required. Special Offer’s discount value, e.g. use 20 for 20% discount.
  • OAuth Scopes:
  • rates_write

Example JSON request

{
  "special_offers": [
    {
      "name": {
        "en": "Special offer for 1"
      },
      "start_at": "2014-08-15T22:42:33Z",
      "end_at": "2014-08-22T22:42:33Z",
      "discount": 10
    }
  ]
}

Response

{
  "links": {
    "special_offers.rental": "https://www.bookingsync.com/api/v3/rentals/{special_offers.rental}"
  },
  "special_offers": [
    {
      "links": {
        "rental": 1
      },
      "id": 1,
      "name": {
        "en": "Special offer for 1"
      },
      "start_at": "2014-08-15T22:42:33Z",

}

Destroy a special offer

Required OAuth scope: :rates_write

Returns empty response with ‘204 No Content’ status code on success.

DELETE /special_offers/:special_offer_id
